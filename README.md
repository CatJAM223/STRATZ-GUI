# Stratz Overlay

**Stratz Overlay** — это приложение-оверлей для Dota 2, которое использует открытое API сервиса Stratz и отображает информацию об игроке в удобном и компактном виде прямо поверх игры.

## Скриншот программы

> Добавьте сюда скриншот главного окна приложения.

---

## Возможности

### Поиск игрока

Поле **Player ID** предназначено для ввода идентификатора игрока. Поддерживаются:

* Steam ID
* ID из клиента Dota 2

После ввода идентификатора нажмите кнопку **Search** для получения информации об игроке через API Stratz.

### Search

Выполняет поиск игрока и загружает его статистику.

### Check API Stratz

Позволяет проверить доступность и работоспособность API Stratz. Если API недоступно или токен указан неверно, приложение уведомит пользователя об ошибке.

### Settings

Открывает окно настроек, в котором можно изменить горячую клавишу для отображения и скрытия оверлея.

По умолчанию используется:

* **F9**

Также поддерживаются комбинации вида:

* Shift + F1
* Shift + F2
* Shift + F3
* и другие аналогичные сочетания.

> **Примечание:** в текущей версии доступны только функциональные клавиши семейства **F1–F12** и их комбинации с **Shift**.

---

## Результат поиска

После ввода ID и нажатия кнопки **Search** отображается карточка игрока со следующей информацией:

* Винрейт (Win Rate)
* Количество сыгранных матчей
* Основная статистика аккаунта
* Дополнительная информация, полученная через API Stratz

# Установка

## 1. Получение токена Stratz

Зарегистрируйтесь на сайте Stratz и получите API-токен.

## 2. Настройка переменных окружения

В каталоге `stratz_api` создайте файл `.env` со следующим содержимым:

```env
STRATZ_TOKEN="ВАШ_ТОКЕН_STRATZ"
DEBUG=True
```

---

## 3. Скачивание проекта

Скачайте или клонируйте репозиторий:

```bash
git clone <URL_РЕПОЗИТОРИЯ>
```

или скачайте архивом и распакуйте его.

---

## 4. Требования

Для работы приложения необходим:

* Python 3.10 или новее

Проверить установленную версию можно командой:

```bash
python --version
```

---

## 5. Создание виртуального окружения

Перейдите в корневую папку проекта и выполните:

```bash
python -m venv .venv
```

После создания окружения активируйте его:

### Windows

```bash
.\.venv\Scripts\activate
```

### Linux/macOS

```bash
source .venv/bin/activate
```

---

## 6. Установка зависимостей

Установите необходимые библиотеки:

```bash
pip install -r requirements.txt
```

---

## 7. Запуск приложения

После успешной установки зависимостей запустите:

```bash
stratz_gui.exe
```

Если все настроено корректно, откроется окно приложения и оверлей будет готов к работе.

---

# Использование

1. Запустите приложение.
2. Введите Steam ID или Dota 2 ID игрока.
3. Нажмите **Search**.
4. Дождитесь загрузки статистики.
5. Используйте горячую клавишу для отображения или скрытия оверлея.
